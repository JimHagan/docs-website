外部サービスインスツルメンテーションでは、Webサービスやクラウド上のリソースなど、プロセス外のサービスへの呼び出しや、その他のネットワーク呼び出しを捕捉します。(外部サービスインスツルメンテーションは、Webサービスやクラウド上のリソース、その他のネットワークコールなど、プロセス外のサービスの呼び出しを捕捉します（MemCacheやデータベース自体など、その他のファーストクラスのバックエンドコンポーネントは含まれません）。

APMの **外部サービス** ページでは、応答時間と1分あたりの外部通話数の上位5つの外部サービスのチャートを提供しています。また、外部サービスのリストは、最も遅い平均応答時間、最も時間のかかるもの、最も高いスループット（1分あたりの要求）でソートすることもできます。当社のUIでは、 [ウェブ外部](/docs/using-new-relic/welcome-new-relic/get-started/glossary#web-external) またはバックグラウンド外部データとも呼ばれています。

アプリのアウトオブプロセスサービスを表示するには、次のようにします。

1. 次のうち1つを行います。

   * New Relic のエクスプローラーです。 **[one.newrelic.com](https://one.newrelic.com) > Explorer> (select an app)> Monitor> External services** にアクセスします。
   * APMです。Go to **[one.newrelic.com](https://one.newrelic.com) > APM> (select an app)> Monitor> External services**.

2. サービスの詳細情報を表示するには、サービス名を選択します。

3. New Relic の標準的な [ユーザーインターフェースの機能](/docs/using-new-relic/user-interface-functions/view-your-data/standard-page-functions) を使って、詳細な情報にドリルダウンすることができます。
