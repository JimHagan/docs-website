.NETエージェントAPIを使用すると、.NETエージェントの機能を拡張することができます。このライブラリを使用するには、 `NewRelic.Api.Agent.dll` への参照をプロジェクトに追加します。また、このパッケージは [NuGet Gallery](https://www.nuget.org/packages/NewRelic.Agent.Api/ "新しいウィンドウにリンクが開きます。") で見ることができます。エージェントを無効にしたり、アンインストールした場合、これらのAPIコールを呼び出しても効果はありません。

特に、APIを使用して、 [アプリ名のカスタマイズ](/docs/agents/net-agent/net-agent-api/setapplicationname-net-agent) 、 [エラーの収集](/docs/agents/net-agent/net-agent-api/noticeerror-net-agent) 、 [カスタム属性の記録](/docs/agents/net-agent/net-agent-api/addcustomparameter-net-agent) 、 [カスタムイベント](/docs/agents/net-agent/net-agent-api/recordcustomevent-net-agent) 、 [カスタムメトリクス](/docs/agents/net-agent/net-agent-api/recordmetric-net-agent) 、 [](/docs/agents/net-agent/net-agent-api/getbrowsertimingheader-net-agent)または [](/docs/agents/net-agent/net-agent-api/disablebrowsermonitoring-net-agent)ブラウザ監視の無効化などを行うことができます。

APIのタスクベースの概要については、 [Guide to using .NET agent API](/docs/agents/net-agent/api-guides/guide-using-net-agent-api) を参照してください。
