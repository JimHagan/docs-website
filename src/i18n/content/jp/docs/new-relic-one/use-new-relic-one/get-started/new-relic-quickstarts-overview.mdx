[New Relic I/O](https://developer.newrelic.com/instant-observability/) は、オープンソースのクイックスタートの豊富なカタログで、統合、ダッシュボード、アラートなどがすぐに使えるようにまとめられています。

各クイックスタートは、世界中のオブザーバビリティの専門家によって作成され、New Relicによって吟味され、ワンクリックでインストールできるようになっています。New Relic I/O は、オブザーバビリティのハブとして、コミュニティの専門知識を活用し、遠隔測定データの価値を高めます。

## 重要な理由 [#why-it-matters]

I/Oカタログでは、お客様のテクノロジースタックのモニタリングを開始するために必要なビルディングブロックをバンドルした数百種類のクイックスタートを選択することができます。このクイックスタートには、インスツルメンテーション、統合、ダッシュボード、アラートなどが含まれており、クリックするだけですぐにインストールできます。

## クイックスタートとは [#what-are-quickstarts]

クイックスタートは、ワンクリックですぐにインストールできるダッシュボード、アラート、インストルメントのバンドルで、当社がサポートする各テクノロジーに特化しています。Node.js、Python、Rubyなどの人気の高いテクノロジーには、フル機能のクイックスタートが用意されていますが、その他のテクノロジーには、インスツルメンテーション、ダッシュボード、アラートが混在しています。

New Relic I/O はオープンソースなので、お客様のニーズに合わせて、既存のクイックスタートを修正・改良したり、新しいクイックスタートを構築したりすることができます。外部からのクイックスタートの編集については、その価値と品質を徹底的に検証しています。コミュニティへの貢献に興味がありますか？ [コントリビューターガイド](https://github.com/newrelic/newrelic-quickstarts/blob/main/CONTRIBUTING.md) をご覧ください。

## いくつかの技術的な詳細 [#technical-detail]

New Relic のクイックスタートでは、オープンソースのインストールレシピを使用して、 [のガイド付きインストールプロセス](docs/full-stack-observability/observe-everything/get-started/new-relic-guided-install-overview) を使って統合を計ることができます。
