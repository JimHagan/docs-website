New Relic の [Azure インテグレーション](/docs/infrastructure/microsoft-azure-integrations/getting-started/introduction-azure-monitoring-integrations) は、インテグレーションに固有のポーリング間隔に従って、Azure サービスにクエリを実行します。

**ポーリング間隔は、統合サービスに関連する** Azure のすべてのエンティティに適用されます。たとえば、13台のCosmosDBインスタンスがある場合、13台のインスタンスのそれぞれが5分ごとにポーリングされます。

## ポーリングデータを見る [#account-status]

Azure インテグレーション [をアクティベートすると、New Relic は Azure からデータのポーリングを開始し、インフラストラクチャ インベントリや New Relic ダッシュボードからデータにアクセスできるようになります。Azure データに加えて、他の New Relic 機能からインポートした追加データを照会することができます。また、特定の統合やアカウント全体のダッシュボードデータを表示することもできます。](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations)

[ポーリング間隔やAPIコールなど、Azureインテグレーションのデータを視覚化します。](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations)

1. [](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations)
2. [](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations)[**one.newrelic.com**](http://one.newrelic.com) **> インフラストラクチャー> Azure** に移動します。
3. 特定の統合のデータを表示するには。インテグレーションの行の「 **Dashboards** 」リンクを選択します。
