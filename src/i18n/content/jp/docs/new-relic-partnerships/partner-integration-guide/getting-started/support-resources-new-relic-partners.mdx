このドキュメントは、New Relic のパートナー様向けです。パートナーシップの統合が完了すると、New Relic の技術担当者が紹介されます。緊急性のないエスカレーションには、これらのチャネルをご利用ください。

<Callout variant="tip">
  New Relic のパートナーでない場合は、New Relic のリソースを参照してください。 [finding help or filing a support ticket](/docs/accounts-partnerships/education/getting-started-new-relic/finding-help) 。
</Callout>

## New Relicサポート [#support]

パートナーアカウントのサポートを受けるには、 [support.newrelic.com](https://support.newrelic.com/) でチケットを作成してください。New Relic に直接お問い合わせいただく場合は、ビジネス開発担当者にご連絡ください。

## ドキュメント [#docs]

[のドキュメント New Relic の docs サイト](https://docs.newrelic.com) は、サポートグループが New Relic の契約者にレベル 1 サポートを提供する際の重要なリソースとなります。これらのリンクをサポートページに掲載することは、セルフヘルプを促し、サポートの労力を軽減する効果的な方法です。

New Relic ドキュメントのトップレベルのエントリーポイント： [docs.newrelic.com](https://docs.newrelic.com).ここから、New Relic の製品や機能に関する情報をカテゴリー別に選択することができます。

<Callout variant="tip">
  docsサイトには、 [Partnerships](/docs/accounts-partnerships/partnerships) というカテゴリーがあり、New Relicのパートナーや一部のパートナーシップのお客様向けの情報が掲載されています。
</Callout>
