[New Relic のブラウザモニタリングエージェントには、](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#agent-types): Lite、Pro、Pro+SPA の 3 種類があります。ブラウザモニタリングを有効にすると、Pro+SPA エージェントがデフォルトでインストールされます。

[Browser Pro feature docs](/docs/browser/new-relic-browser/browser-pro-features) では、Pro および Pro+SPA エージェントで利用できる機能について説明しています。詳しくは、 [エージェントの種類](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#agent-types) をご覧ください。
