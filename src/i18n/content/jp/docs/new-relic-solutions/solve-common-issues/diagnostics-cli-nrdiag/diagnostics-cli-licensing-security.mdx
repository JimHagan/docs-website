他のNew Relicツールと同様に、Diagnostics CLIサービスは、お客様とお客様の顧客のデータプライバシーを保護するように設計されています。Diagnostics CLIは、New Relic製品の構成や操作性を評価する診断チェックを行うために関連するシステム情報やNew Relic製品の成果物(ログや設定ファイル)を検査します。

デフォルトでは、このデータは New Relic に送信されません。この情報を HTTPS でサポートチケットにアップロードするオプションはあります。New Relic のセキュリティ対策の詳細については、 [セキュリティとプライバシーに関するドキュメント](/docs/security/security-privacy/data-privacy/data-privacy-new-relic) をご覧いただくか、 [New Relic のセキュリティウェブサイト](https://newrelic.com/security) をご覧ください。

## ライセンス契約 [#diagnostics-licenses]

Diagnostics CLI の使用は、本 [使用許諾契約書](/docs/licenses/license-information/other-licenses/new-relic-diagnostics-licenses) に加え、Diagnostics CLI で使用されるオープンソース・ソフトウェアの使用許諾契約書に従うものとします。

## アカウントにアップロードされたデータ [#uploaded-account-data]

Diagnostics CLIでは、特定のコマンドライン引数を使用すると、この情報をHTTPS経由でサポートチケットにアップロードすることができます。機密情報が含まれていると思われるファイルを収集する前に、プロンプトが表示されます。

`nrdiag-output.zip` と `nrdiag-output.json` に含まれる収集ファイル [が New Relic にアップロードされる前に、お客様にもプロンプトが表示されます。これにより、提供したくない情報を確認・編集することができます。(例えば、 `nrdiag-output.zip` にはお客様のユーザー名が含まれます)。また、アップロードを完全にキャンセルすることもできます。](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/interpret-nrdiag-output)

## [データストレージ](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/interpret-nrdiag-output)

[`nrdiag` を使用して作成された、または `nrdiag` のデータを含む、20MB 以下のサポートチケットの添付ファイルは、すべて Zendesk に保存されます。これらの添付ファイルはすべて New Relic に保存されます。詳細については、 ](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/interpret-nrdiag-output)[Zendesk のプライバシーポリシーおよびデータ保護ポリシー](https://www.zendesk.com/company/privacy-and-data-protection/) を参照してください。

## 環境変数

診断 CLI は、以下の環境変数を調べて診断チェックを行います。これらの変数の値は、ローカルで `nrdiag-output.json` ファイルに記録されます。これらは以下の通りです。

* `NEWRELIC` または `NEW_RELIC を含むすべての環境変数`
* `NRIA で始まるすべての環境変数`

**A-C:**

* `APP_ENV`
* `APPDATA`
* `cor_enable_profiler`
* `COR_PROFILER`
* `cor_profiler_path`
* `coreclr_enable_profiling`
* `coreclr_profiler`
* `coreclr_profiler_path`

**D-I**

* `default_locale_cfg_file`
* `docker_api_version`
* `DOCKER_HOST`
* `dotnet_install_path`
* `dotnet_sdk_version`
* `GLIBC_REPO`
* `GLIBC_VERSION`
* `HOME`

**J-L:**

* `JAVA_HOME`
* `JAVA_JCE`
* `JAVA_PACKAGE`
* `java_version_build`
* `java_version_major`
* `java_version_minor`
* `JBOSS_HOME`
* `LANG`
* `LOCALAPPDATA`

**M:**

* `minion_api_endpoint`
* `minion_api_proxy`
* `minion_api_proxy_self_signed_cert`
* `minion_check_timeout`
* `minion_docker_api_version`
* `minion_docker_host`
* `minion_docker_runner_apparmor`
* `minion_docker_runner_registry_endpoint`
* `MINION_GROUP`
* `MINION_JAR`
* `MINION_JVM_MB`
* `minion_jvm_opts`
* `minion_log_level`
* `minion_provider`
* `MINION_USER`

**N-Z**

* `PATH`
* `プログラムファイル`
* `プログラムデータ`
* `RACK_ENV`
* `RAILS_ENV`
* `RUBY_ENV`
* `作業ディレクトリ`
