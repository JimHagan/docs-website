New Relic CodeStream を Microsoft Teams チャンネルに接続します。

コードマークを投稿すると、チームメイトにはアクティビティフィードで通知され、メールでも通知されます。しかし、時には、Microsoft Teamsにも共有したいことがあるかもしれません。そうすれば、まだCodeStreamに参加していない人や、IDEにあまり時間をかけていない人にも情報を届けることができるでしょう。

## Microsoft Teamsへの接続 [#connect]

1. [ここをクリックしてCodeStreamアプリをインストールする](https://teams.microsoft.com/l/app/7cf49ab7-8b65-4407-b494-f02b525eef2b?source=store-copy-link) または、Microsoft Teamsのサイドバーにあるアプリに行き、CodeStreamを検索します。

2. インストールすると、CodeStreamアプリの以下のようなポップアップが表示されます。 **Add** ボタンをクリックします。注意：このボタンを展開して、チームやチャットに追加するオプションを選択しないでください。単に「追加」をクリックします。 ![アプリの追加](/images/MSTCSAppPage.png)

3. そうすると、CodeStreamボットとのプライベートチャットになり、TeamsのサイドバーにCodeStreamのロゴが表示されています。このチャットで `signin` と入力して送信します。 ![サインインコマンド](/images/MSTSigninMsg.png)

4. ボットからの投稿が、 **Sign in** ボタンとともに、ストリームに表示されます。 ![サインインボタン](/images/MSTSigninButton.png)

5. ボタンをクリックすると、ウェブに表示され、CodeStreamへのサインインを促されます。 ![CodeStreamにサインインします。](/images/MSTCSSigninPage.png)

6. サインインすると、トークンが渡されます。このトークンをコピーして、MS Teamsに戻ってCodeStreamボットとのチャットに貼り付ける必要があります。

   ![シグニントークン](/images/MSTSigninToken.png)

7. トークンをチャットに貼り付けて送信します。

   ![トークンメッセージ](/images/MSTTokenMsg.png)

8. 共有したいチャンネルに行き、「 `@` 」と入力し、ポップアップで「 **Get bots** 」を選択します。

   ![ボットの取得](/images/MSTGetBots.png)

9. リストからCodeStreamボットを選択し（必要であれば検索してもよい）、 **Add** をクリックします。

   ![ボットの追加](/images/MSTAddABot.png)

10. チャンネルに戻り、 `connect` コマンドでCodeStream botを@mentionします。これを共有したい任意のチャンネルで繰り返します。

    ![コネクト・ボット](/images/MSTConnectBot1.png)

11. 以下のような確認が取れれば、CodeStreamからTeamsに共有する準備が整います。

    ![アプリのアップロード](/images/MSTConnected1.png)

12. CodeStreamに戻ると、先ほど接続したチャンネルに共有できるようになっています。

    ![チャンネル選択](/images/MSTSelectChannel.png)

## Microsoft TeamsからCodeStreamに参加する [#participate]

Microsoft Teamsに共有すると、チームメイトにコードマークが通知されるだけでなく、チームメイトはそのコードマークを使って自分のIDEに直接ジャンプし、CodeStream上の会話に参加することができます。

![チームで共有](/images/ShareOnMST1.png)

### IDEで開く [#open-ide]

**IDEで開く** をクリックすると、コードとディスカッションの両方をお使いのIDEの中で見ることができます。CodeStream の Web ページが表示され、どの IDE を開くかを選択できます。CodeStream は選択した内容を記憶しており、次回同じリポジトリのディスカッションを閲覧する際にはその内容が反映されます。次に、あなたの IDE の適切なソースファイルに移動し、関連するコードブロックにスクロールし、ディスカッションが CodeStream ペインに表示されます。IDE でそのリポジトリを開いていない場合は、CodeStream が自動的にソースファイルを開きます (CodeStream がインストールされている状態で、以前にそのリポジトリを開いていたと仮定していますので、私たちはその場所を知っています)。

JetBrainsのIDEを使用している場合は、CodeStreamがIDEにディープリンクできるように、 [Toolbox app](https://www.jetbrains.com/toolbox-app/) をインストールしてください。

### GitHub（またはBitbucket、GitLab）での公開 [#open-github]

コードブロックがGitHub、Bitbucket、またはGitLabでホストされているリポジトリのものである場合、このボタンはそのホスティングサービス上の対応するコードブロックに移動します。
