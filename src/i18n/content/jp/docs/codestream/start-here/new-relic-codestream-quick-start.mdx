以下のクイックステップに従って、New Relic と CodeStream の使用を開始してください。

<Callout title="プレビューリリース">
  CodeStream の New Relic One との統合は、当社の米国データセンターの New Relic One アカウントに限定されたプレビューリリースであり、お客様の使用はプレリリースポリシーに従うものとします。(これは他のすべてのCodeStreamの機能には適用されません)。
</Callout>

## New Relicのアカウントを取得する [#account]

New Relicアカウントがない場合、[newrelic.com/signup](https://newrelic.com/signup)でサインアップしてください。永久無料です。

## New RelicによるCodeStreamのインストール [#quickstart]

[CodeStream quickstart](https://developer.newrelic.com/instant-observability/codestream/29bd9a4a-1c19-4219-9694-0942f6411ce7) を使って、お使いのIDEにインストールしてください。

### CodeStreamをIDEにインストールします。 [#install-ide]

CodeStreamは、Visual Studio Code、Visual Studio extensionsメニュー、またはJetBrains pluginsメニューからもインストールできます。

## New Relicのインストール [#install-new-relic]

[Add your data](https://one.newrelic.com/launcher/nr1-core.settings?pane=eyJuZXJkbGV0SWQiOiJ0dWNzb24ucGxnLWluc3RydW1lbnQtZXZlcnl0aGluZyJ9) UIページの手順に従って、データを流し込みます。
